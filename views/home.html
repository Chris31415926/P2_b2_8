English

This website is purely for academic purposes. Any and all data found on this website is fabricated and should not be taken seriously.

If you have stumbled upon this website we encourage you to not answer the questions truthfully or take the questionnaire seriously since:

  • There is no guarantee that the data can be protected aginst users with ill intentions.
  • There has not been contact with medical personnel about using the website, so your answers will not be seen by proper personnel.
  • The website has been made in connection to a project trying to solve the problem of finding people who doesn't know they have a chronic disease.
    This project is made by software students studying at Aalborg University.
    The website is therefore only a resource used in the project and should not be seen as a standalone thing.
    For this to happen, the website needs a lot of improvements.
